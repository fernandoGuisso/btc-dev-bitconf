Testnet 3

é uma alternativa ao block chain Bitcoin. Isso permite que desenvolvedores e bitcoin testers experimentem, sem ter que usar bitcoins reais ou se preocupar em quebrar a cadeia principal do Bitcoin.

Why 3 ?!?!

Em resumo: A Testnet foi usada como Trade, o que dificultou a utilizanção para testes. A Testnet 2 entrou com um novo genesis block. Por fim a Testnet 3 chegou resolvendo alguns problemas e com novas features, em cima de um terceiro genesis block.

Principais Diferenças

  • Protocolo de rede e RPC em outra porta. (18333 e 18332)
  • ADRESSVERSION diferente para não funcionar na mainnet. (0X6F ao inves de 0X00)
  • Dificuldade de mineirar blocos na mainnet é metade da dificuldade na tesnet.

Explorando o Blockchain

Explorar o Blockchain, simplesmente é consultar os blocos para saber informações guardadas nele.

Cada bloco tem essas informações:

  • Uma lista de transações
  • Hash para o bloco anterior
  • Tamanho do bloco
  • Contagem de transações
  • Block Header

Online

  • Block Cypher

  • Blockchain.info
  • Blockr

Gerando um address

const bitcore = require('bitcore-lib');
const privateKey = new bitcore.PrivateKey(bitcore.Networks.testnet);
const publicKey = new bitcore.PublicKey(privateKey);
const address = new bitcore.Address(publicKey, bitcore.Networks.testnet);

URI's e QRCodes

URI scheme

O objetivo do esquema URI é permitir que os usuários façam pagamentos facilmente clicando em links em páginas da web ou digitalizando códigos QR.

URI scheme

bitcoin:[address][?amount=][?label=][?message=]
bitcoin:175tWpb8K1S7NmH4Zx6rewF9WQrcZv245W
bitcoin:175tWpb8K1S7NmH4Zx6rewF9WQrcZv245W?label=Luke-Jr
bitcoin:175tWpb8K1S7NmH4Zx6rewF9WQrcZv245W?amount=20.3&label=Luke-Jr
